# PureTask Category Playbooks

Purpose: break `docs/puretask-decisions.md` into small execution guides by category.

How to use:
- Find the category relevant to your task.
- Apply its "Build Rules" while implementing.
- Complete its "Done Checklist" before PR.
- Run `always-follow-workflow.md` for the full end-to-end process.
- Start each task with `feature-kickoff-template.md`.
